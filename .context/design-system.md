# Design System - LazyTrip

## Stack Tecnol√≥gico

- **Framework:** TailwindCSS v3
- **Iconos:** Lucide React (Pendiente de instalar si se requiere)

## Paleta de Colores

### Primary (Acciones Principales)

- **Base:** `bg-blue-600` (Sky/Blue)
- **Hover:** `hover:bg-blue-700`
- **Text:** `text-white`

### Feedback

- **Error:** `text-red-500`, `bg-red-50`
- **Success:** `text-green-600`

## Componentes Base (Referencias)

### Botones

Usa estas clases cuando crees botones:
`<button className="px-4 py-2 rounded bg-blue-600 text-white font-medium hover:bg-blue-700">Texto</button>`

### Inputs

Usa estas clases cuando crees formularios:
`<input className="w-full p-2 border rounded-md shadow-sm focus:ring-2 focus:ring-blue-500" />`

## Layout (Estructura)

- **Navbar:** Sticky (pegada arriba), 64px de altura.
- **Container:** Siempre usar `mx-auto px-4` para centrar el contenido.
